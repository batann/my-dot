## *Installation**

1. Clone the repository
2. Assign permission to install.sh
3. execute install.sh
4. source .bashrc

``` sh
git clone https://github.com/batann/my-dot
sudo chmod +x my-dot/install.sh
sudo bash my-dot/install.sh
source ~/.bashrc
```
